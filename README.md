# B Lord Bot v3.0

Автоматизированный бот для игры Beast Lord: The New Land

## Установка

1. Установить зависимости:
```bash
pip install -r requirements.txt
```

2. Запустить GUI:
```bash
python main_gui.py
```

## Требования

- Windows OS
- LDPlayer 9
- Python 3.8+
